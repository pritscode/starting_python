# 딕셔너리(dictionary)
리스트(list)가 '인덱스를 기반으로 값을 저장하는것'이라면 딕셔너리는 '키(key)를 기반으로 값을 저장하는 것.'
## 딕셔너리 선언
딕셔너리는 중괄호 {}로 선언. 키: 값 형태를 쉼표(,)로 연결해서 만듬. 키는 문자열, 숫자, 불 등으로 선언 가능. 일반적으로 문자열 사용.
```
변수 = {
  키: 값,
  키: 값,
  ...
}

>>> dict_a = {
      "name": "어벤저스 엔드게임",
      "type": "히어로 무비"
      }
```
## 딕셔너리 요소 접근
```
>>> dict_a
{'name': '어벤저스 엔드게임', 'type': '히어로 무비'}
```
특정 키만 따로 출력 가능. 요소에 접근할 때는 딕셔너리 뒤에 대괄호[]를 입력하고 내부에 인덱스처럼 키를 입력.딕셔너리를 선언할 때는 중괄호{}를 사용하지만 딕셔너리 요소에 접근할 때는 리스트처럼 딕셔너리 뒤에 대괄호[]를 입력 후, 인덱스처럼 키를 입력.
```
>>> dict_a
'어벤저스 엔드게임'
```
딕셔너리 내부의 값에 문자열, 숫자, 불 등의 다양한 자료를 넣을 수도 있음. 리스트와 딕셔너리도 하나의 자료. 리스트와 딕셔너리를 값으로 넣을 수도 있음.
```
>>> dict_b = {
      "director": ["안소니 루소", "조 루소"],
      "cast": ["아이언맨", "타노스", "토르", "닥터스트레인지", "헐크"]
      }
>>> dict_b
{'director': ['안소니 루소', '조 루소'], 'cast': ['아이언맨', '타노스', '토르', '닥터스트레인지', '헐크']}
>>> dict_b["director"]
['안소니 루소', '조 루소']
```
```
# 딕셔너리 선언
dictionary = {
  "name": "7D 건조 망고",
  "type": "당절임",
  "ingredient": ["망고", "설탕", "메타중아황산나트륨", "치자황색소"],
  "origin": "필리핀"
}

# 출력
print("name: ", dictionary["name"])
print("type: ", dictionary["type"])

# 값 변경
dictionary["name"] = "8D 건조 망고"
print("name: ", dictionary["name"])

# 실행 결과
# name: 7D 건조 망고
# type: 당절임
# name: 8D 건조 망고
```
ingredient는 dictionary의 키이기도 하지만, 여러 개의 자료를 가지고 있는 리스트. 인덱스를 지정하여 리스트 안의 특정 값을 출력 가능.
```
>>> dictionary["ingredient"]
['망고', '설탕', '메타중아황산나트륨', '치자황색소']
>>> dictionary["ingredient"][1]
'설탕'
```
파이썬은 딕셔너리의 키에 따옴표 없이 단순하게 식별자를 입력하면 이를 변수로 인식. name이라는 이름을 변수로 만들어 주면 됨. type은 type() 함수라는 기본 식별자가 있기 때문에 이것이 키로 들어가 따로 오류 발생하지는 않음.
```
>>> name = "이름"
>>> dict_key = {
      name: "7D 건조 망고",
      type: "당절임"
      }
>>> dict_key
{'이름': '7D 건조 망고', <class 'type'>: '당절임'}
```

## 딕셔너리에 값 추가/제거
딕셔너리[새로운 키] = 새로운 값
```
>>> dictionary["price"] = 5000
>>> dictionary
{'name': '8D 건조 망고', 'type': '당절임', 'ingredient': ['망고', '설탕', '메타종아황산나트륨', '치자황색소'], 'origin': '필리핀', 'price': 5000}    # "price" 키 추가.
>>> dictionary["name"] = "8D 건조 파인애플"
{'name': '8D 건조 파인애플', 'type': '당절임', 'ingredient': ['망고', '설탕', '메타종아황산나트륨', '치자황색소'], 'origin': '필리핀', 'price': 5000}
>>> del dictionary["ingredient"]
>>> dictionary
{'name': '8D 건조 파인애플', 'type': '당절임', 'origin': '필리핀', 'price': 5000}
```

## 딕셔너리 내부에 키 있는지 확인
### in 키워드
```
# 딕셔너리 선언
dictionary = {
  "name": "7D 건조 망고",
  "type": "당절임",
  "ingredient": ["망고", "설탕", "메타중아황산나트륨", "치자황색소"],
  "origin": "필리핀"
}

# 사용자로부터 입력 받음
key = input("> 접근하고자 하는 키: ")

# 출력
if key in dictionary:
  print(dictionary[key])
else:
  print("존재하지 않는 키에 접근하고 있습니다.")

# 실행 결과
# > 접근하고자 하는 키: name
# 7D 건조 망고
# > 접근하고자 하는 키: ㅇㅂㅇ
# 존재하지 않는 키에 접근하고 있습니다.
```
### get() 함수
문자열이 내부적으로 많은 함수를 갖고 있는 것을 확인. 딕셔너리도 뒤에 .(마침표)를 찍고 자동 완성 기능으로 확인하면 다양한 기능이 있다는 것을 알 수 있음.  
키의 존재와 관련된 get() 함수만 살펴봄. get() 함수는 딕셔너리의 키로 값을 추출하는 기능. 딕셔너리[키]를 입력할 때와 같은 기능 수행. 존재하지 않는 키에 접근할 경우 None을 출력.
```
# 딕셔너리 선언
dictionary = {
  "name": "7D 건조 망고",
  "type": "당절임",
  "ingredient": ["망고", "설탕", "메타중아황산나트륨", "치자황색소"],
  "origin": "필리핀"
}

# 존재하지 않는 키에 접근
value = dictionary.get("존재하지 않는 키")
print("값: ", value)

# None 확인 방법
if value == None:
  print("존재하지 않는 키에 접근했었습니다.")

# 실행 결과
# 값: None
# 존재하지 않는 키에 접근했었습니다.
```

## for 반복문: 딕셔너리
for 반복문과 딕셔너리를 조합해서 사용. 주의할 점은 딕셔너리 내부에 있는 키가 변수에 들어감.
```
# 딕셔너리 선언
dictionary = {
  "name": "7D 건조 망고",
  "type": "당절임",
  "ingredient": ["망고", "설탕", "메타중아황산나트륨", "치자황색소"],
  "origin": "필리핀"
}

# for 반복문 사용
for key in dictionary:
  # 출력
  print(key, ":", dictionary[key])

# 실행 결과
# name : 7D 건조 망고
# type : 당절임
# ingredient : ['망고', '설탕', '메타중아황산나트륨', '치자황색소']
# origin : 필리핀
```
